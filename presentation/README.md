# Presentation Assets

- `slides.md`: Markdown slide deck (Marp-compatible).
- `slides.html`: Standalone HTML slide deck (open directly in a browser).

## Quick Start

1. Open `presentation/slides.html` in a browser.
2. Use Left/Right arrow keys to navigate.

## Optional Rendering (Marp)

If you use Marp CLI:

```bash
marp presentation/slides.md --html -o presentation/slides-rendered.html
marp presentation/slides.md --pdf -o presentation/slides.pdf
```
